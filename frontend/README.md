## ğŸš€ **Quick Start**

### Prerequisites
- Node.js 16+ 
- npm or yarn

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/yourusername/hugup.git
cd hugup
```

2. **Install dependencies**
```bash
npm install
# or
yarn install
```

3. **Start development server**
```bash
npm start
# or
yarn start
```

4. **Open your browser**
Visit `http://localhost:3000` to see Hugup in action!
